❓ Guía de uso

1️⃣Busca el producto escribiendo su nombre arriba, o toca 📷 para escanear el código del cajón.
2️⃣Toca + para sumar una unidad, o − para restar.
3️⃣¿Sabes la cantidad exacta? Toca el número del centro y escríbela directo con el teclado.
4️⃣Cuando termines, toca 📊 Exportar Excel para guardar el conteo.
💡 Toca ✅ Actualizados para ver solo los productos que ya contaste y no perder el hilo.
1️⃣Toca los productos que necesitan etiqueta — se marcan en verde al seleccionarlos.
2️⃣¿Necesitas etiquetar toda una categoría? Toca el filtro de esa categoría y luego Sel. cat.
3️⃣Elige el tamaño: Peq = más etiquetas por hoja · Grd = etiquetas más grandes y fáciles de leer.
4️⃣Toca 🖨 Ver e imprimir y confirma la impresión en la pantalla que aparece.
💡 El número sobre el ícono 🏷 te dice cuántos productos tienes listos para imprimir.
📥Importar productos: sube un archivo Excel o CSV con la lista de productos del almacén.
🔄Reemplazar todo: borra la lista actual y carga la del archivo nuevo. Úsalo cuando el catálogo cambie completamente.
➕Agregar / Actualizar: añade productos nuevos o corrige los existentes sin borrar el resto.
🗑️Limpiar conteos: borra todas las cantidades del conteo actual. La lista de productos no se toca.
💣Restaurar catálogo: vuelve a la lista original de 161 productos y borra los conteos. Úsalo solo si algo salió muy mal.
⚠️ Las acciones de Limpiar y Restaurar no se pueden deshacer. Úsalas con cuidado.
🔎No encuentro el producto: prueba escribir solo las primeras letras, o escanea el código QR del cajón.
↩️Me equivoqué en la cantidad: busca el producto y vuelve a tocar el número para corregirlo.
🔢El número no me deja escribir: toca el número con firmeza (no el + ni el −) y espera el teclado.
📵La cámara no abre: acepta el permiso de cámara cuando el navegador lo pida, o busca por nombre.
⚙️ Para cambiar la lista de productos o ajustes avanzados, habla con el administrador.
